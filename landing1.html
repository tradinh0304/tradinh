Stock Market Analysis

Developed a Python-based tool leveraging the Yahoo Finance API to extract and analyze real-time stock market data, providing insights into recent market trends and performance metrics.

Introduction

The Yahoo Finance API is an application programming interface that provides access to financial market data from Yahoo Finance. Through this API, developers and users can retrieve real-time and historical stock data, as well as other financial metrics such as: Stock prices (open, close, high, low) Trading volumes Historical data for specific time periods Company financials like revenue, earnings, and more Market summaries for indices, commodities, currencies, and ETFs sthat allow for stock analysis, portfolio tracking, or forecasting market trends. The API makes it easier to pull in stock market data and integrate it into dashboards, websites, or analytical models.

Focusing on

As a student with a strong passion for finance and stock analysis, I am keen on developing tools that can assist in making informed investment decisions. This project aims to visualize stock data for major financial institutions such as JP Morgan, Bank of America, Wells Fargo, and Goldman Sachs using the Yahoo! Finance database. By analyzing the stocks of these selected companies, I seek to gain insights that can help in the construction of a well-balanced, personalized investment portfolio. Through this initiative, I aim to deepen my understanding of financial markets and enhance my skills in portfolio management
